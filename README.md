##  Featured Project

### 🔹 CIS API — Ideas & Topics Management System
> *Academic capstone project — Software Development 3*

A full REST API for managing topics, ideas and a voting system, built with a professional architecture in a team environment.

**Tech:** `C#` `ASP.NET Core` `Java` `MySQL` `Entity Framework` `AutoMapper` `FluentValidation`

**Architecture & Patterns:**
-  Clean Architecture (4-layer separation)
-  Domain-Driven Design (DDD) — aggregates, entities, repository interfaces
-  CQRS — separate Command/Query services and controllers
-  SOLID principles applied throughout
-  Dependency Injection via IoC container
-  Test coverage: unit, integration and controller tests with mocks

**Project Management:**
- 24 issues tracked on GitLab with mandatory code review
- 95% of features delivered (23/24 issues completed)
- Labels, milestones and clear assignees per task
